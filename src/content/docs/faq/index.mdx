## What is Markdown?

Markdown is a lightweight plain-text format that uses simple syntax for structure, emphasis, links, images, code, and lists.

For serious work, that short definition is not enough. The more useful definition is that Markdown is a source format that stays readable as text while still being transformable into HTML and other outputs.

## What is MarkdownPic?

MarkdownPic is a Markdown reference site plus a companion Markdown-to-image tool at [tool.markdownpic.com](https://tool.markdownpic.com/).

## Is MarkdownPic only about screenshots?

No. The site is broader than the tool. It covers syntax, editors, publishing systems, workflows, and Markdown operations.

## Is Markdown the same everywhere?

No. CommonMark defines a core standard, but GitHub Flavored Markdown and many renderers add or change behavior. That is why a Markdown file can behave differently across platforms.

## When should I use Markdown-to-image?

Use it when the deliverable is a visual asset, not when the HTML page needs to be the primary searchable source.

## When should I not use Markdown-to-image?

Do not use image export as a replacement for a searchable HTML page when the primary goal is discovery, crawlability, and long-term search value. Use it as a distribution layer when the channel rewards visual packaging more than HTML rendering.
