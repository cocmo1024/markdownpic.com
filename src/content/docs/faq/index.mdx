## What is Markdown?

Markdown is a lightweight plain-text format that uses simple syntax for structure, emphasis, links, images, code, and lists.

## What is MarkdownPic?

MarkdownPic is a Markdown reference site plus a companion Markdown-to-image tool at [tool.markdownpic.com](https://tool.markdownpic.com/).

## Is MarkdownPic only about screenshots?

No. The site is broader than the tool. It covers syntax, editors, publishing systems, workflows, and Markdown operations.

## When should I use Markdown-to-image?

Use it when the deliverable is a visual asset, not when the HTML page needs to be the primary searchable source.
