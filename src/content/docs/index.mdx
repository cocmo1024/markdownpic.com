## How to use MarkdownPic

<Steps>
	1. Start with the guide or syntax page that matches the exact Markdown problem you are trying to solve.
	2. Move into editor or publishing comparisons only after the core content boundary is clear.
	3. Use workflow pages when Markdown needs to support AI reuse, screenshots, docs, or multi-channel distribution.
	4. Open the tool when the output needs to become a long image, polished screenshot, or visual asset.
</Steps>
