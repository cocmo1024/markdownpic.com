# MarkdownPic

Starlight-based documentation site for `markdownpic.com`.

## Focus

- Markdown syntax and portability
- Editor, workflow, and publishing decisions
- Markdown-to-image and AI-era Markdown operations

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Use Cloudflare Pages or Worker static assets with:

- Build command: `npm run build`
- Output directory: `dist`
