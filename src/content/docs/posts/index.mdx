This section preserves evergreen post URLs that already exist or are intentionally kept as post-style assets.

The site is moving toward a structured Markdown encyclopedia, but some post-style URLs already have direct search value or link history. Those URLs should stay live, useful, and internally connected instead of being abandoned.

## What belongs here

Posts belong here when:

- the URL already exists and should not break,
- the content is still useful to search visitors,
- or the topic reads better as an article than as a reference node.

## What does not belong here

Thin trend posts, disposable commentary, and shallow roundups do not belong here. If a page cannot carry evergreen search value or be upgraded into something durable, it should not exist.
